
# 🚗 BYD BladeBot – Discord Sales & Community Bot

[![Discord.js](https://img.shields.io/badge/discord.js-v14-blue.svg)](https://discord.js.org)
[![Node.js](https://img.shields.io/badge/node.js-v18%2B-green.svg)](https://nodejs.org)
[![PostgreSQL](https://img.shields.io/badge/postgresql-15-blue.svg)](https://www.postgresql.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**BYD BladeBot** is a full-featured Discord bot for BYD car sales lead generation, community engagement, and automated marketing. It combines an AI-powered sales assistant, ticket system, verification, giveaways, auto-poster, and lifelike lobby chatter – all managed through an intuitive admin dashboard.

---

## ✨ Features

### 🚗 Lead Generation & Sales
- **15 BYD models** (Seagull, Dolphin, Seal, Seal Performance, ATTO 3, Tang, Song Plus, Yuan Plus, Han, Han Performance, Yangwang U8, Yangwang U9, Commercial, eBus, eTruck)
- **USD pricing** with federal/state EV tax credits
- **Instant quotes** with breakdown of incentives
- **Test drive booking** with interactive calendar picker
- **Trade-in estimator** with multi-step modal flow
- **Personalised advisor** names and 1-hour response promise
- **Lead scoring system** (HOT → WARM → INTERESTED → AWARE → COLD)

### 🎫 Support & Verification
- **Ticket system** – users create private tickets with category selection
- **Verification system** – one-click role assignment with questions
- **Logs channel** for tracking all support activity
- **Staff role management** with permission levels

### 🤖 Automated Content
- **Auto Poster** – AI-generated BYD news, tips, model spotlights (OpenRouter API)
- **Lobby Chatter** – 15+ personas with 750+ conversation snippets simulating realistic chat
- **Testimonial posts** – automated review sharing
- **Follow-up scheduler** – re-engage inactive leads

### 🎁 Giveaways
- **Regular giveaways** – any prize, random winner selection
- **Car Giveaways** – win a BYD car, admin selects winners (can select ANYTIME)
- **Entry tracking** with private channels per entrant
- **Payment status tracking** for winners

### 🎛️ Admin Dashboard
- All configurations in one place (`/admin`)
- Buttons for verification, tickets, auto poster, lobby chatter, giveaways
- Per-server settings stored in PostgreSQL
- Detailed statistics and health monitoring

### 🖼️ Media & Branding
- Static image serving via Express
- Custom banners, logos, and icons
- Professional BYD-themed embeds
- UI Avatars API for dynamic avatars

### 📊 Database
- PostgreSQL persistence
- Lead tracking, test drive bookings, tickets, giveaways
- User state management with caching
- Interaction analytics tracking

## 📦 Installation

### Prerequisites

- Node.js v18 or higher
- PostgreSQL database (local or cloud – Render, Supabase, Neon)
- Discord Bot Token ([Discord Developer Portal](https://discord.com/developers/applications))
- OpenRouter API Key ([OpenRouter](https://openrouter.ai))

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/steadybb/EMBEB-BOT/
   cd byd-bladebot
```

1. Install dependencies
   ```bash
   npm install
   ```
2. Configure environment variables
   Create a .env file in the root directory:
   ```env
   # Discord
   DISCORD_TOKEN=your_discord_bot_token
   CLIENT_ID=your_application_id
   BOT_OWNER_IDS=123456789,987654321
   
   # Database
   DATABASE_URL=postgresql://user:password@localhost:5432/bydbot
   DB_POOL_MAX=20
   DB_IDLE_TIMEOUT=30000
   DB_CONNECT_TIMEOUT=10000
   
   # AI API
   OPENROUTER_API_KEY=your_openrouter_key
   
   # Web Server
   PORT=3000
   STATIC_BASE_URL=https://your-domain.onrender.com/static
   
   # Auto Poster
   AUTO_POST_CHANNELS=123456789,987654321
   AUTO_POST_SCHEDULE=0 */2 * * *
   AUTO_POST_ALL_CHANNELS=true
   AUTO_POST_ON_STARTUP=true
   AUTO_POST_MIN_DELAY=5000
   
   # Lobby Chatter
   LOBBY_CHATTER_SCHEDULE=*/3 * * * *
   LOBBY_WEBHOOK_URL=https://discord.com/api/webhooks/...
   
   # Logging
   LOG_LEVEL=info
   LOG_TO_FILE=false
   LOG_TIMESTAMP=true
   LOG_EMOJI=true
   LOG_COLOR=true
   LOG_SEPARATE_ERRORS=true
   ```
3. Run database migrations (auto-creates on first start)
4. Deploy slash commands
   ```bash
   npm run deploy
   ```
5. Start the bot
   ```bash
   npm start
   ```

🚀 Deployment on Render

1. Push your code to GitHub
2. Create a new Web Service on Render
3. Connect your repository
4. Add all environment variables (from .env)

· Create all database tables
· Register slash commands
· Start all schedulers (follow-up, auto poster, lobby chatter, testimonial)

🎯 Usage Guide

First-Time Setup

1. Invite the bot to your server with required permissions (Manage Channels, Manage Roles, Send Messages, Embed Links, Use Slash Commands)
2. Open Admin Dashboard: /admin
3. Configure Verification:
   · Set a role: /verify role @Verified
   · Set channel (optional): /verify channel #verification
   · Enable: /verify enable
   · Post panel: /verify setup
4. Configure Ticket System:
   · Set category: /ticket category #Tickets
   · Set staff role: /ticket staffrole @Support
   · Set logs channel: /ticket logs #ticket-logs
   · Post panel: /ticket setup
5. Configure Auto Poster:
   · Go to admin dashboard → Auto Poster
   · Set channels (channel IDs)
   · Enable auto posting
6. Configure Lobby Chatter:
   · Create a webhook in your lobby channel
   · Go to admin dashboard → Lobby Chatter
   · Set webhook URL
   · Enable lobby chatter
7. Start a Giveaway:
   · Regular: /giveaway start prize:"BYD Hoodie" duration:24 winners:2
   · Car: /cargiveaway start model:Seal shipping:1999 duration:168 winners:1
