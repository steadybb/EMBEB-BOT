
# 🚗 BYD BladeBot – Discord Sales & Community Bot

[![Discord.js](https://img.shields.io/badge/discord.js-v14-blue.svg)](https://discord.js.org)
[![Node.js](https://img.shields.io/badge/node.js-v18%2B-green.svg)](https://nodejs.org)
[![PostgreSQL](https://img.shields.io/badge/postgresql-15-blue.svg)](https://www.postgresql.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)

**BYD BladeBot** is a full-featured Discord bot for BYD car sales lead generation, community engagement, and automated marketing. It combines an AI-powered sales assistant, ticket system, verification, giveaways, auto-poster, and lifelike lobby chatter – all managed through an intuitive admin dashboard.

## ✨ Features

### 🚗 Lead Generation & Sales
- **14 BYD models** (Seagull, Dolphin, Seal, Seal Performance, ATTO 3, Tang, Song Plus, Yuan Plus, Han, Han Performance, Yangwang U8, Yangwang U9, Commercial, eBus)
- **USD pricing** with federal/state EV tax credits
- **Instant quotes** with breakdown of incentives
- **Test drive booking** with interactive calendar picker
- **Trade-in estimator** with multi-step modal flow
- **Personalised advisor** names and 1-hour response promise

### 🎫 Support & Verification
- **Ticket system** – users create private tickets, staff closes them
- **Verification system** – one-click role assignment
- **Logs channel** for tracking all support activity

### 🤖 Automated Content
- **Auto Poster** – AI-generated BYD news, tips, model spotlights every 2 hours (OpenRouter API)
- **Lobby Chatter** – 30+ personas with 500+ conversation snippets simulating realistic chat every 2 minutes via webhook

### 🎁 Giveaways
- **Regular giveaways** – any prize, winner selection, reroll support
- **Car Giveaways** – win a BYD car (winner pays shipping + documentation fee)

### 🎛️ Admin Dashboard
- All configurations in one place (`/admin`)
- Buttons for verification, tickets, auto poster, lobby chatter, giveaways
- Per-server settings stored in PostgreSQL

### 🖼️ Media & Branding
- Static image serving via Express
- Custom banners, logos, and icons
- Professional BYD-themed embeds

### 📊 Database
- PostgreSQL persistence
- Lead tracking, test drive bookings, tickets, giveaways
- User state management with caching

## 📦 Installation

### Prerequisites

- Node.js v18 or higher
- PostgreSQL database (local or cloud – Render, Supabase, Neon)
- Discord Bot Token ([Discord Developer Portal](https://discord.com/developers/applications))
- OpenRouter API Key ([OpenRouter](https://openrouter.ai))

### Setup

1. **Clone the repository**
   ```bash
   git clone https://github.com/yourusername/byd-bladebot.git
   cd byd-bladebot
```

1. Install dependencies
   ```bash
   npm install
   ```
2. Configure environment variables
   Create a .env file in the root directory:
   ```env
   DISCORD_TOKEN=your_discord_bot_token
   CLIENT_ID=your_application_id
   DATABASE_URL=postgresql://user:password@localhost:5432/bydbot
   OPENROUTER_API_KEY=your_openrouter_key
   STATIC_URL=https://your-domain.onrender.com/static
   AUTO_POST_CHANNELS=123456789,987654321
   LOG_TO_FILE=false
   ```
3. Run database migrations
   The bot automatically creates tables on first start.
4. Deploy slash commands
   ```bash
   npm run deploy
   ```
5. Start the bot
   ```bash
   npm start
   ```

🎯 Usage Guide

First-Time Setup

1. Invite the bot to your server with the required permissions (Manage Channels, Manage Roles, Send Messages, Embed Links, Use Slash Commands)
2. Run /admin – you'll see the dashboard with all systems
3. Configure Verification:
   · Set a role: /verify role @Verified
   · Enable: /verify enable
   · Post panel: /verify setup
4. Configure Ticket System:
   · Set category: /ticket category #Tickets
   · Set staff role: /ticket staffrole @Support
   · Post panel: /ticket setup
5. Configure Auto Poster:
   · Set channels: via admin dashboard → Auto Poster → Set Channels
   · Enable: via admin dashboard
   · Content will post every 2 hours automatically
6. Configure Lobby Chatter:
   · Create a webhook in your lobby channel
   · Set webhook URL: via admin dashboard → Lobby Chatter → Set Webhook URL
   · Enable: via admin dashboard
   · Conversations will start every 2 minutes
7. Start a Giveaway:
   · Regular: /giveaway start prize:"BYD Hoodie" duration:24 winners:2
   · Car: /cargiveaway start model:Seal shipping:1999 duration:168

📁 Project Structure

```
byd-bladebot/
├── commands/           # Slash commands (admin, giveaway, cargiveaway, ticket, verify, posttemplate)
├── events/             # Discord events (interactionCreate, guildMemberUpdate)
├── handlers/           # Interaction handler
├── modules/            # Embed templates (bydEmbeds)
├── schedulers/         # Cron jobs (followUp, autoPost, lobbyChatter)
├── utils/              # Utilities (database, logger, bydData, stateManager, calendar, buildEmbed, lobbyChatter, helpers, openai)
├── static/             # Static images for embeds
├── keepalive.js        # Express server for Render uptime
├── deploy-commands.js  # Slash command registration script
├── index.js            # Main entry point
├── package.json        # Dependencies
└── .env                # Environment variables (not committed)
```
